🛡️ Security Features

🔐
End-to-End Encryption
AES-256-GCM encryption using SubtleCrypto API. Keys are derived using ECDH key exchange — only participants can read messages. Keys never leave your device.
👤
Zero Identity
No account, email, or phone required. You're assigned a random anonymous name. No IP logging, no tracking, no cookies.
⏱
30-Second Vanish Mode
When enabled, messages self-destruct after 30 seconds. Togglable per session. Messages also clear when you close the tab.
🚫
No Persistence
All messages exist only in memory. Nothing is stored to disk, databases, or localStorage. Closing the tab = total erasure.
🔏
Message Integrity
Each message includes a tamper-evident hash. Modified messages are flagged and rejected.
