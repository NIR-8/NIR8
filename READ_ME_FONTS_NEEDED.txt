Two font files still needed in this folder for the math/equation styling to work:

1. poppins-v24-latin-500.woff2
   - From: https://gwfh.mranftl.com/fonts/poppins?subsets=latin
   - Select weight 500, download, take the .woff2 file, keep this exact name.

2. cmu-serif-roman.woff2
   - From: https://cdn.jsdelivr.net/gh/vsalvino/computer-modern@main/fonts/CMUSerif-Roman.woff2
   - Download this file directly, then RENAME it to: cmu-serif-roman.woff2

3. cmu-serif-italic.woff2
   - From: https://cdn.jsdelivr.net/gh/vsalvino/computer-modern@main/fonts/CMUSerif-Italic.woff2
   - Download this file directly, then RENAME it to: cmu-serif-italic.woff2

Put all three .woff2 files directly in this assets/fonts/ folder (no subfolders).
